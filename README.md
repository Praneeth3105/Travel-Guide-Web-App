📌 Project Overview

The Travel Guide Web Application is a web-based platform designed to help users explore travel destinations and book trips easily. The application provides a user-friendly interface with form validations and an email-based booking confirmation system using PHPMailer.
It is developed using HTML, CSS, PHP, and runs on a XAMPP local server.

🛠️ Technologies Used

Frontend: HTML5, CSS3
Backend: PHP
Server: XAMPP (Apache & MySQL)
Email Service: PHPMailer
Database: MySQL
Tools: Visual Studio Code (VS Code), Google Chrome / Microsoft Edge

✨ Features

🌐 Responsive travel guide website
🧾 User-friendly booking form
✅ Client-side and server-side form validation
📧 Booking confirmation via email using PHPMailer
🗃️ Secure storage of booking details in the MySQL database
🔒 Input validation to prevent invalid data submission
